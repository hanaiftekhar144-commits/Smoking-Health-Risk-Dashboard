# Smoking Health Risk Analysis Dashboard

This project is a Power BI dashboard analyzing the impact of smoking on health risks.

## Key Insights
- Smoking shows a strong correlation with increased health risks
- Age group 49–58 shows peak risk levels
- Damaged patients have higher BMI and cholesterol on average

## Tools Used
- Power BI
- Data Analysis
- Data Visualization
